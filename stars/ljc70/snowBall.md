snowBall
========

> 基础组件拖拽生成组合组件的工具，最终将生成的组件保存到指定目录。

Build Setup
-----------

# 进入展示项目下载依赖
cd static/vue-tmp && npm i

# 回退到当前目录下载依赖
cd .. && cd .. && npm i

# 启动项目
npm run dev

# 项目打包
npm run build
