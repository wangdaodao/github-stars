daily-poetry-image
------------------

每天一句中国古诗词，生成 AI 图片

### 👉🏽 Live Demo https://we-drawing.com

项目介绍
----

-   本项目是使用 `stabilityai/stable-diffusion-3-5-large` 模型生成的图片，每天一句中国古诗词作为 Prompt。
-   GitHub Action 自动触发。
-   诗词由今日诗词提供 API。
-   网站使用 Astro 构建。
-   使用硅基流动文生图 API

设置 `SILICON_FLOW_TOKEN`
-----------------------

在 GitHub 项目设置里，设置 Sceret `SILICON_FLOW_TOKEN`，值为硅基流动 Token，请自行获取。

文章（旧版本）
