daily-poetry-image
------------------

每天一句中国古诗词，生成 AI 图片 Powered by Bing DALL-E-3.

### 👉🏽 Live Demo https://we-drawing.com

项目介绍
----

-   本项目是基于 Bing Image DALL-E-3 生成的图片，每天一句中国古诗词作为 Prompt。
-   GitHub Action 自动触发。
-   诗词由今日诗词提供 API。
-   网站使用 Astro 构建。
