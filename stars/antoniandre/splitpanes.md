Splitpanes
==========

> A Vue.js reliable, simple and touch-ready panes splitter / resizer. Vue 3 compatible.

Installation
------------

**Vue 3**

```
npm i splitpanes
```

**Vue 2**

```
npm i splitpanes@legacy
```
