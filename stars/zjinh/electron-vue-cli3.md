🖥 运行效果
-------

### 克隆代码

git clone https://github.com/zjinh/electron-vue-cli3.git

### 安装依赖

cd electron-vue-cli3
npm i

### 淘宝的npm源

npm config set registry 'https://registry.npm.taobao.org'
npm config set ELECTRON\_MIRROR='https://npm.taobao.org/mirrors/electron/'

或者使用Yarn安装依赖

### 运行项目

npm run dev
npm run dev:web

### 打包项目

npm run build
npm run build:web
