young-cli
=========

功能
--

-   css预编译
-   js预编译
-   路由加载模式
-   单页面/多页面
-   jsx支持

安装
--

npm

```
npm install -g young-cli 
```

yarn

```
yarn global add young-cli
```

命令行工具
-----

-   create -创建新项目

-   serve -启动本地开发服务
-   build -启动打包服务
