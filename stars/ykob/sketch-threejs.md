Requirements
------------

Node >=12.x <15

Usage
-----

1.  Install the necessary node modules.

```
npm i
```

1.  Run the npm script `start` to start development.

```
npm run start
```

1.  When finishing development, run the npm script `build` to make the compressed files.

```
npm run build
```
