wq-cli
======

安装
--

```
$ npm install -g cynthiawupore/wq-cli
```

使用
--

```
$ wq vue
$ wq react
```

功能说明
----

```
$ wq vue

初始化脚手架 https://github.com/cynthiawupore/wq-vue
```

```
$ wq react

初始化脚手架 https://github.com/cynthiawupore/wq-react
```
