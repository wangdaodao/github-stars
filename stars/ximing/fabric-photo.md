fabric photo
============

基于 canvas 的纯前端的图片编辑器，支持方形，圆形，箭头，缩放，拖拽，鹰眼，马赛克，涂鸦，线条，导出 png，剪切等

online Demo
-----------

访问 https://ximing.github.io/fabric-photo/ 体验

启动 demo
-------

# 安装依赖
npm run i
# 运行项目
npm run dev

结合 react 使用方法
